# Eczema Detection using CNN

This project uses a Convolutional Neural Network (CNN) to classify skin images into eczema and normal categories.

## Features

* Image classification using deep learning
* Data augmentation for improved generalization
* Achieved ~95% accuracy

## Technologies

* Python
* TensorFlow / Keras
* Google Colab

## Dataset

* Eczema and normal skin images

## Future Work

* Integration with wearable sensor data (EDA, temperature, motion)
* Multi-modal eczema severity prediction system
